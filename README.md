# Annual Developer Goals

## Get Started

Ensure you have Node `v20.x` installed

```sh
node --version
# expected: v20.x
```

Install dependencies

```sh
npm i
```

Create a new goals/reflections template

```sh
bin/new <type: goals|reflections> [year]
```

Navigate to the new <type> file and begin!
